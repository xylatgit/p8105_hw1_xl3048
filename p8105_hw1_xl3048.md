# Problem 1

Create a data frame comprised of specific varible types.

``` r
tibble(
  samp=rnorm(10),
  samp_gt_0=samp > 0,
  char_vec=c("a","b","c","d","e","f","g","h","i","j"),
  factor_vec=factor(c("low","low","low","mod","mod","mod","mod","high","high","high"))
)
```
